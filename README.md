Logininfo Project

A simple PHP login system with MySQL database storage and styled frontend.

Features

User login form with username, password, and remember me checkbox,
Fancy frontend with video background and styled UI.

Folder Structure

Logininfo/
├── loginpage.html # Login form
├── loginpage.php # PHP script to process login
└── README.md # Project documentation

Open the project in your local server (XAMPP / WAMP / LAMP) and test login.
